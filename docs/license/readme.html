版本说明

版本说明

UBSI的系统构成主要包括三个方面：

  • 核心组件

    包括UBSI基础框架、Container容器、Consumer以及基础的命令行工具

  • 部署工具、网关及基础服务

    包括命令行部署工具、Restful-API网关以及日志、仓库等基础微服务

  • 治理工具

    集成的Web应用，可视化监控、配置UBSI的运行环境

UBSI核心框架

UBSI核心框架分为"社区版"和"正式版"，两个版本的差异如下：

社区版 正式版
是否开源 开源，Apache-2.0 license 否
使用费用 无 免费
获取方式 https://github.com/ubsice https://ubsi-home.github.io
服务支持 开源社区 用户社区
功能限制 - 不支持单例服务的自动切换
- 不支持分布式事务		 无

部署工具、网关和基础服务

UBSI的命令行部署工具、Restful-API服务网关以及各种基础微服务不区分发行版本：

说明
是否开源 即将
使用费用 免费
获取方式 https://ubsi-home.github.io
服务支持 用户社区
功能限制 无

治理工具

UBSI治理工具分为"开发版"和"企业版"，两个版本的差异如下：

开发版 企业版
是否开源 否 否
使用费用 免费 付费
获取方式 https://ubsi-home.github.io https://ubsi-home.github.io
服务支持 用户社区 专项技术支持
功能限制 - 不支持redis/mongodb的容错部署
- 不支持脚本/调度/编排/事务4个微服务的多实例部署		 无

治理工具的两个版本在代码上完全一致，只不过"开发版"不需要license授权文件，可以在开发/测试环境中一直免费使用，如果在正式生产环境中部署UBSI治理工具，需要申请license，将"开发版"升级为"企业版"，可以获得完整的高可用部署方案以及专门的技术支持服务。

申请"企业版"license请与我们联系：ubsi@rewin.com.cn
